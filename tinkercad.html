plataforma arduino


Arduino é uma plataforma que possibilita o desenvolvimento de projetos eletrônicos. O Arduino é constituído de hardware e software, tornando assim possível a realização de diversos projetos tecnológicos.


minicontrolador atmel Arduino


O microcontrolador tem o papel fundamental de receber os dados vindos do sensor, interpretá-lo e enviar o resultado para o display.Este microcontrolador é o responsável pela forma transparente como funciona a placa Arduino Uno, possibilitando o upload do código binário gerado após o programa feito pelo usuário.
