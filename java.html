O que é o Java?


O Java é uma linguagem de programação popular orientada a objetos, multiplataforma e centrada em rede que pode ser usada como uma plataforma em si. O java é uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor.
